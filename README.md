## Getting Started

### Prerequisites

- TypeScript
- npm

### Installation

1. Clone the repository:

```bash
git clone https://github.com/fantastizeey1/Invoice-app
```

2. Navigate to the project:

```bash
cd Invoice-app
```

3. Install dependencies:

```bash
npm install
```

## Usage

Run the project:

```bash
npm start
```

## Testing

Run the test suite:

```bash
npm test
```
